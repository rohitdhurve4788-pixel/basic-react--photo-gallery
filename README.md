# 📸 Photo Gallery - React

A simple and responsive Photo Gallery project built using React.js.

This project fetches random images from an API and displays them in a beautiful gallery layout. Users can generate new images by clicking the button.

## 🚀 Features

- Generate random images
- Fetch images using API
- Responsive photo gallery layout
- Dynamic image height for gallery effect
- Clean UI using Tailwind CSS
- Smooth hover animation

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- Axios
- Tailwind CSS
- Picsum Photos API
- Lucide React Icons
